# Finance OS - Projet de Vision Stratégique

## Présentation
Finance OS est une plateforme EPM (Enterprise Performance Management) AI-Native conçue pour disrupter les outils legacy (Tagetik, SAP, Oracle). 

## Objectifs du projet
- Définir la vision produit d'un Finance Lakehouse.
- Documenter l'architecture conceptuelle (AI-first, souveraineté Mistral).
- Préparer le plan d'exécution pour le MVP.

## Structure du dossier
- `.cursorrules` : Directives de comportement pour l'IA (Expert Finance).
- `01_VISION_STRATEGIQUE.md` : Le manifeste fondateur du projet.
- `02_ARCHITECTURE_CONCEPTUELLE.md` : Détails techniques et flux de données.
- `03_OBJECTIONS_ET_REPONSES.md` : Argumentaire face aux DAF et DSI.

**Note : Ce dépôt est actuellement en phase de conception stratégique (Zéro Code).**
